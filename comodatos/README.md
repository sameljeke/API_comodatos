# Sistema de Gestión de Comodatos - API REST

API REST para la gestión de comodatos de instrumentos musicales, desarrollada con Flask y MariaDB.

## Características

- ✅ Autenticación JWT con refresh tokens
- ✅ Autorización basada en roles (admin, representante, invitado)
- ✅ CRUD completo para todas las entidades
- ✅ Validación y sanitización de datos
- ✅ Rate limiting y protección CORS
- ✅ Exportación a Excel/CSV
- ✅ Importación desde Excel
- ✅ Dashboard con estadísticas y alertas
- ✅ Documentación Swagger/OpenAPI
- ✅ Pruebas unitarias
- ✅ Logging completo
- ✅ Migraciones de base de datos
